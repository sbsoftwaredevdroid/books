Camera Usage

« FindMe » only uses the device camera on user’s request. In order to use the camera, a permission is needed. Starting at Android 6.0 Marshmallow, users can choose to decline this permission. Before Android 6.0 Marshmallow, users have to accept the permission before installing the application, according to Google and Android policies. The camera is only used to take a picture that will be referred as an information for help user to find his car or favorite location.

Coarse Location / Fine Location

« FindMe » only uses this permission to allow the user to share a position if he wishes or save a position locally on his smartphone (to easily find his car for example)

Data Collected

« FindMe » gathers anonymous usage data in order to improve the quality of the service.

The data is processed within Firebase and allows the developer to monitor wich features are used or not in order to improve « BibliBooks ». The anonymous data is not used for any commercial purpose and remains at the executive use of the developer.
